# ProxyMaster v1.0.0 首次发布

## 🎉 欢迎使用 ProxyMaster

ProxyMaster 是一个现代化的Chrome代理管理扩展程序，旨在替代已停止维护的Proxy SwitchyOmega，为用户提供更强大、更智能的代理管理体验。

## ✨ 核心功能

### 🔧 代理管理
- **多协议支持**: HTTP、HTTPS、SOCKS4、SOCKS5
- **配置管理**: 创建、编辑、删除代理配置
- **快速切换**: 一键切换不同代理配置
- **连接测试**: 实时测试代理连接状态

### 🤖 智能切换
- **自动切换规则**: 基于域名、URL、通配符、正则表达式
- **规则优先级**: 灵活的规则优先级管理
- **自动回退**: 连接失败时自动回退到直连
- **实时匹配**: 页面加载时即时应用规则

### 🌍 国际化支持
- **多语言界面**: 中文简体、英文、日文、韩文
- **自动检测**: 根据浏览器语言自动切换界面
- **完整本地化**: 所有界面元素和消息都已本地化

### 📊 性能监控
- **请求统计**: 实时监控网络请求数量
- **切换统计**: 记录代理切换次数
- **性能分析**: 代理连接性能数据

### 🔄 数据管理
- **配置导出**: 导出所有配置和规则
- **配置导入**: 从备份文件恢复配置
- **数据同步**: Chrome账户同步配置数据

## 🚀 技术特性

### Manifest V3

- 更好的安全性和性能

### 现代化架构
- Service Worker后台服务
- 响应式用户界面

### 智能优化
- 快速规则匹配算法
- 最小化网络延迟
- 优化的资源使用

## 📱 用户界面

### 弹出窗口
- 简洁直观的代理状态显示
- 快速切换代理配置
- 实时统计信息展示

### 选项页面
- 完整的配置管理界面
- 可视化规则编辑器
- 详细的设置选项

### 右键菜单
- 快速切换代理

- **本地存储**: 所有数据存储在本地

## 📋 系统要求

- Chrome 88+ 或基于Chromium的浏览器
- 支持Manifest V3的浏览器版本

## 🛠️ 安装方法

### Chrome Web Store（推荐）
1. 访问Chrome Web Store
2. 搜索"ProxyMaster"
3. 点击"添加至Chrome"

### 开发者模式安装
1. 下载扩展程序文件
2. 打开Chrome扩展管理页面 (`chrome://extensions/`)
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择ProxyMaster文件夹

## 📖 使用指南

### 快速开始
1. 点击扩展图标打开弹出窗口
2. 点击"新建配置"创建代理配置
3. 填写代理服务器信息
4. 保存并切换到新配置

### 自动切换设置
1. 打开选项页面
2. 切换到"自动切换"标签
3. 点击"添加规则"
4. 设置匹配条件和目标代理
5. 启用自动切换功能
